# لعبتي Premium

## الوصف
لعبة بسيطة بـ Pygame فيها Player وEnemies وGems.

## التشغيل
- لتشغيل اللعبة على الكمبيوتر:
  - شغل `main.exe` لو انت عامل EXE
  - أو افتح CMD وجرب:
    ```bash
    python main.py
    ```
- جميع الملفات موجودة داخل الريبو.

## ملاحظات
- اللعبة شغالة على Windows.
