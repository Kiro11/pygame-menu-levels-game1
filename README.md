# MyGame

A simple 2D game developed using Python and Pygame.

## Game Idea
The player moves around the screen to collect gems while avoiding enemies.
Each collected gem increases the score. Colliding with an enemy ends the game.

## Features
- Player movement using keyboard
- Enemy obstacles
- Score tracking
- Game over condition

## Technologies
- Python
- Pygame

## How to Run
1. Install Python.
2. Install Pygame:
   pip install pygame
3. Run the game:
   python main.py
